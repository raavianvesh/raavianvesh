### 🚀 3. Collab-Verse (Distributed Web3 & Microservices Ecosystem)
* **Core Stack:** `Java (Reactive)`, `Spring Boot`, `Spring Security`, `React`, `TypeScript`, `Solidity`, `Apache Kafka`, `PostgreSQL` , `Flyway`
* **Architectural Paradigms & Patterns:**
  - **Domain-Driven Design (DDD):** Modeled complex business boundaries across segregated contexts (Identity, Wallet, Ingestion).
  - **Hexagonal Architecture (Ports & Adapters):** Isolated the core business domain from external framework dependencies, databases, and blockchain RPC clients.
  - **12-Factor App Methodology:** Built for cloud-native compliance (strict config-to-environment separation, stateless processes, and fast disposability).
  - **Object-Oriented Design (OOD) & GoF Patterns:** Applied fundamental structural and behavioral patterns (Factory, Strategy, Observer) to manage complex execution flows dynamically.
* **Component Architecture:**
  - **Reactive Event Streaming:** Built non-blocking asynchronous data pipelines using **Reactive Java** and distributed backpressure orchestration via **Apache Kafka**.
  - **Security & Identity:** Implemented granular token-based authentication and service-to-service validation frameworks leveraging **Spring Security**.
  - **Decentralized Services:** Written secure **Solidity Smart Contracts** to manage wallet states and decentralized identity routing rules.
  - **Full-Stack Interface:** Designed a modular, type-safe single-page application using **React**, **TypeScript**, and highly optimized custom **SCSS** styling patterns.

### 🎙️ 4. VoiSys (Voice Processing Infrastructure)
* **Core Stack:** `C#`, `.NET Core`, `Google Cloud SDK (Speech-to-Text)`, `NAudio`, `Asynchronous Programming`
* **What it does:** An enterprise-focused background infrastructure utility that captures native system audio streams and handles real-time distributed ingestion pipelines for semantic transcription.
* **Key Achievements:**
  - Engineered low-latency audio stream processing pipelines in **C#** utilizing the **.NET Core** runtime environment.
  - Handled memory management protocols for buffering raw PCM audio data packets dynamically before secure network streaming.
  - Implemented async/await tasks to achieve seamless integration with the **Google Cloud SDK** for speech recognition without blocking host execution layers.
